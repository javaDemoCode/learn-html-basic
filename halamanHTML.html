Disini ingin belajar memahami struktur HTML dan tag pada html serta kegunanaan nya apa saja itu berikut yang saya ketahui:
  • P : tag P itu digunakan untuk membuat paragraf
  • H1-H6 : tag-tag ini digunakan untuk membuat heading dengan tingkat kepentingan yang berbeda
  • a : tag ini digunakan untuk membuat tautan link, biasanya dia selalu di isi dengan href untuk menghubungkan ke halaman lain
  • title : tag ini digunakan untuk membuat judul halaman biasanya terdapat di atas tag body atau dalam tag head
  • body : tag ini biasanya berisi konten utama dari halaman web jadi apapun yang di tulis di dalam tag ini maka akan tampil di halaman web
  • head : tag ini berisi informasi meta tentang halaman web, bisa juga berisikan title, meta tags, link:css, script:src, dll
  • ul : tag ini digunakan untuk membuat daftar yang tidak berurutan, biasanya di dalamnya terdapat tag li untuk membuat item-item dalam daftar
  • li : tag ini digunakan untuk membuat item dalam daftar, biasanya digunakan di dalam tag ul atau ol
  • bold : tag ini digunakan untuk mempertebal suatu kata yang di bungkus di dalam tag tersebut
  • italic : tag ini digunakan untuk memiringkan suatu kata yang di bungkus di dalam tag tersebut
  • img : tag ini digunakan untuk menyisipkan gambar ke dalam halaman web, biasanya di isi dengan atribut src untuk menentukan sumber gambar dan alt untuk memberikan deskripsi alternatif jika gambar tidak dapat ditampilkan
  • button : tag ini digunakan untuk membuat tombol yang dapat diklik
  • style : tag ini digunakan untuk menambahkan gaya CSS ke dalam halaman web
